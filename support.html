Common Setup Issues

The script won't run?

Ensure you have the latest version of Node.js installed. Run `npm install` inside the script directory before execution.

MX records aren't resolving?

Check if your server's firewall allows outgoing requests on Port 25 (SMTP) or if your ISP blocks it. Some cloud providers (like AWS or DigitalOcean) block this by default.
